# Changelog - Controle de Armada

## [1.0.0] - 2024-07-13

### Adicionado
- Criação de competições com data e quantidade de armadas configuráveis
- Cadastro de participantes ilimitado com nome e número opcional
- Registro rápido de resultados (Praia/Nado) com navegação automática
- Salvamento automático de todos os dados
- Banco de dados local com Hive
- Sistema de undo/redo completo
- Estatísticas em tempo real (contagem e percentual)
- Exportação para PDF
- Exportação para Excel
- Temas claro e escuro com Material Design 3
- Tela inicial com 3 opções (Nova, Continuar, Salvas)
- Tela de planilhas salvas com ações (Continuar, Renomear, Duplicar, Excluir)
- Interface intuitiva com botões grandes
- Animações suaves
- Suporte a múltiplas plataformas (Android, iOS, Windows)
- Preparação para Firebase

### Melhorias
- Design responsivo
- Performance otimizada
- Código limpo e bem documentado
- Tratamento de erros robusto
- Validations de entrada
